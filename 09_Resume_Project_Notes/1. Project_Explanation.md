# SmartHire QA Project Explanation

SmartHire is a web-based recruitment platform tested as an end-to-end QA project.

The project included testing of:
- Authentication module
- Candidate workflows
- Recruiter workflows
- Interview scheduling
- Admin functionalities

Testing activities performed:
- Test scenario creation
- Test case design
- Bug reporting
- API testing using Postman
- RTM preparation
- Functional and regression testing

Tools used:
- Excel
- Postman
- GitHub

The project helped in understanding real-world QA workflows and defect management processes.

# SmartHire QA Project

SmartHire is a web-based recruitment platform tested as an end-to-end QA project.

## Modules Covered
- Authentication
- Candidate Module
- Recruiter Module
- Interview Scheduling
- Admin Module

## Testing Performed
- Functional Testing
- UI Testing
- Regression Testing
- Smoke Testing
- API Testing

## Project Deliverables
- Test Plan
- Test Scenarios
- Test Cases
- Bug Reports
- RTM
- API Testing Documentation
- Test Summary Report

## Tools Used
- Excel
- Postman
- GitHub

## Key Highlights
- Designed 39+ test cases
- Reported multiple defects
- Performed basic API testing
- Created RTM and QA documentation

## AI Usage in Project

AI tools were used during the project for:
- Test scenario brainstorming
- Test case improvement
- Bug description refinement
- QA documentation guidance
- Learning API testing workflows

All testing activities, validations, API execution, GitHub setup, and project organization were performed manually.
